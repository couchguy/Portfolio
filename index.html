Electronic Chess Board

This was my Senior Design Project that I worked to complete with four other Computer Engineering students. Our goal of the project was to create a chess board that would interface with a chess Engine ("Tom's Simple Chess Program" in our case) to allow a game of physical chess to be played with only one person. The technologies we used in our project were a BeagleBone Black running the chess engine and game logic communicating to an Arduino that controlled the IR sensors and LEDs to indicate and detect moves.

Camera Tracker

In a class called Embedded Systems III we had a course long project of using 2 servo motors, a camera, and an ATMega32 board to track a white square on a black piece of paper. The trick of this project was that the ATMega32 does not have enough memory to hold a full picture to use for comparison. The comparison need to be done on the fly as the picture was being read in.
